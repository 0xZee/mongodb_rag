# ChatBot
Streamlit ChatBot app to RAG into MongoDB Atlas :
- Inference & LLM : `Groq Inference`, Model : `llama3.2 3b`
- Embeddings : `Cohere`
- Vector Store : `MongoDB Atlas Vector Store`

```python
pip install -r requirements.txt
streamlit run app.py
```
